# LinkedIn Reel Prompts

## Category

video-prompts

## Recommended Use

Create professional short-form video scripts and shot lists for LinkedIn.

## Ideal For

- PM tips
- AI workflows
- Dashboard explainers
- Construction insights

## Compatible Tools

- ChatGPT
- Runway
- Pika
- Canva

## Level

intermediate

## Variables

- `[TOPIC]`
- `[AUDIENCE]`
- `[DURATION]`
- `[HOOK]`
- `[CALL_TO_ACTION]`

## Prompt

```text
Create a LinkedIn reel about [TOPIC] for [AUDIENCE]. Duration: [DURATION]. Hook: [HOOK]. Call to action: [CALL_TO_ACTION]. Provide the hook, scene-by-scene structure, on-screen text, voiceover, visual direction, and caption. Make it concise, useful, and professional.
```

## Example Use

```text
Create a LinkedIn reel about three early warning signs in project cost control.
```

## Expected Output

A short-form video script with clear hook, visual structure, and caption.

## Notes

Use one core idea per reel.
