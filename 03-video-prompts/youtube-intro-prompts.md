# YouTube Intro Prompts

## Category

video-prompts

## Recommended Use

Create professional YouTube intro concepts for educational, technical, or executive channels.

## Ideal For

- AI education channels
- Project management videos
- Power BI tutorials
- Construction project content

## Compatible Tools

- ChatGPT
- Runway
- Pika
- Canva

## Level

intermediate

## Variables

- `[CHANNEL_TOPIC]`
- `[AUDIENCE]`
- `[VISUAL_STYLE]`
- `[DURATION]`
- `[TAGLINE]`

## Prompt

```text
Create a YouTube intro concept for a channel about [CHANNEL_TOPIC]. Audience: [AUDIENCE]. Visual style: [VISUAL_STYLE]. Duration: [DURATION]. Tagline: [TAGLINE]. Provide visual sequence, motion direction, music mood, on-screen text, and voiceover. Make it memorable, premium, and short enough to respect the viewer time.
```

## Example Use

```text
Create a YouTube intro concept for a channel about AI for project managers.
```

## Expected Output

A concise intro concept with branding, motion, and copy direction.

## Notes

Keep intros short and quickly transition into value.
