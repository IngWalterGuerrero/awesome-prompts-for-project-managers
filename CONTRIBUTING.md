# Contributing

Thank you for helping improve Walter AI Prompt Library.

## File Naming

Use lowercase kebab-case file names:

```text
executive-board-presentation.md
schedule-analysis-prompts.md
power-bi-dashboard-visuals.md
```

## Prompt Quality Standards

Each prompt should:

- Solve a clear use case.
- Include practical variables where the user should customize the prompt.
- Explain the expected output.
- Mention compatible tools when relevant.
- Avoid vague phrasing such as "make it better" without defining what better means.
- Prefer reusable structures over one-time examples.

## Required Prompt Structure

Use [prompt-template.md](prompt-template.md) as the starting point for every new prompt.

At minimum, include:

- Title
- Category
- Recommended use
- Ideal for
- Compatible tools
- Variables
- Prompt
- Expected output
- Notes

## Category Guidance

- Use `01-slide-prompts` for presentation and deck generation.
- Use `02-image-prompts` for static visuals and image generation.
- Use `03-video-prompts` for video generation, reels, intros, and cinematic transitions.
- Use `04-llm-prompts` for ChatGPT, Claude, Gemini, and general assistant workflows.
- Use `05-powerbi-prompts` for Power BI, DAX, dashboards, and data modeling.
- Use `06-construction-projects` for construction and infrastructure project workflows.
- Use `07-project-management` for general PM processes and delivery management.
- Use `08-business-strategy` for executive, commercial, and strategic prompts.

## Review Checklist

Before contributing, check that:

- The prompt has a clear outcome.
- Variables are easy to replace.
- The prompt can be reused in more than one scenario.
- The file name matches the prompt topic.
- The category is correct.
