# Strategy Memo

## Category

business-strategy

## Recommended Use

Create a concise strategy memo for executive decision-making.

## Ideal For

- Strategic options
- New initiatives
- Market positioning
- Investment decisions

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

advanced

## Variables

- `[STRATEGIC_QUESTION]`
- `[CONTEXT]`
- `[OPTIONS]`
- `[TRADEOFFS]`
- `[RECOMMENDATION]`

## Prompt

```text
Write a strategy memo answering this question: [STRATEGIC_QUESTION]. Context: [CONTEXT]. Options: [OPTIONS]. Tradeoffs: [TRADEOFFS]. Recommendation: [RECOMMENDATION]. Use an executive memo structure: situation, decision required, options, tradeoffs, recommendation, risks, and next steps. Be concise and direct.
```

## Example Use

```text
Write a strategy memo about whether to launch an AI prompt library for project managers.
```

## Expected Output

A clear strategy memo with decision framing, options, tradeoffs, recommendation, and next steps.

## Notes

Best used when the decision and available options are already defined.
