# Executive Status Report

## Category

project-management

## Recommended Use

Create concise executive project status reports focused on decisions, risks, progress, and next steps.

## Ideal For

- Weekly updates
- Steering committees
- Sponsor communication
- Portfolio reporting

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

intermediate

## Variables

- `[PROJECT_NAME]`
- `[PERIOD]`
- `[PROGRESS]`
- `[RISKS]`
- `[DECISIONS_NEEDED]`
- `[NEXT_STEPS]`

## Prompt

```text
Create an executive status report for [PROJECT_NAME]. Period: [PERIOD]. Progress: [PROGRESS]. Risks: [RISKS]. Decisions needed: [DECISIONS_NEEDED]. Next steps: [NEXT_STEPS]. Write it for senior stakeholders. Be concise, clear, and action-oriented. Separate facts, risks, decisions, and recommendations.
```

## Example Use

```text
Create an executive status report for the North Substation Expansion project.
```

## Expected Output

A concise executive update with clear project status, risks, decisions, and next actions.

## Notes

Add traffic-light status only if the criteria are clearly defined.
