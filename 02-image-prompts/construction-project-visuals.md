# Construction Project Visuals

## Category

image-prompts

## Recommended Use

Generate realistic or premium conceptual visuals for construction project communication.

## Ideal For

- Project updates
- Infrastructure presentations
- Risk communication
- Site progress content

## Compatible Tools

- Midjourney
- DALL-E
- ChatGPT

## Level

intermediate

## Variables

- `[PROJECT_TYPE]`
- `[SCENE]`
- `[TIME_OF_DAY]`
- `[STYLE]`
- `[ASPECT_RATIO]`

## Prompt

```text
Create a professional construction project visual for a [PROJECT_TYPE]. Scene: [SCENE]. Time of day: [TIME_OF_DAY]. Style: [STYLE]. Aspect ratio: [ASPECT_RATIO]. Make it realistic, technically credible, and suitable for executive project communication. Show scale, safety, progress, and coordination without making the scene feel chaotic.
```

## Example Use

```text
Create a professional construction project visual for a high-voltage substation during commissioning.
```

## Expected Output

A credible construction visual that can support project reporting, presentations, or stakeholder communication.

## Notes

Specify region, equipment, PPE, and project phase for better realism.
