# Power BI Dashboard Visuals

## Category

image-prompts

## Recommended Use

Create visual concepts for Power BI dashboards before designing the actual report.

## Ideal For

- Dashboard mockups
- Executive analytics
- Data storytelling
- Portfolio reporting

## Compatible Tools

- ChatGPT
- Midjourney
- DALL-E

## Level

intermediate

## Variables

- `[DASHBOARD_TOPIC]`
- `[AUDIENCE]`
- `[KPIS]`
- `[STYLE]`
- `[FORMAT]`

## Prompt

```text
Create a premium Power BI dashboard visual concept for [DASHBOARD_TOPIC]. Audience: [AUDIENCE]. Key KPIs: [KPIS]. Style: [STYLE]. Format: [FORMAT]. Design the dashboard with strong information hierarchy, clear KPI cards, useful trend visuals, meaningful filters, and executive-level readability. Avoid decorative charts and low-value visuals.
```

## Example Use

```text
Create a premium Power BI dashboard visual concept for construction project cost control.
```

## Expected Output

A dashboard design brief with layout, visual types, KPI hierarchy, and style direction.

## Notes

Use this before building the report to clarify what decisions the dashboard should support.
