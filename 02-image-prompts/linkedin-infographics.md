# LinkedIn Infographics

## Category

image-prompts

## Recommended Use

Generate polished infographic concepts for LinkedIn posts about project management, AI, construction, or analytics.

## Ideal For

- LinkedIn carousels
- Thought leadership
- Educational posts
- PM frameworks

## Compatible Tools

- ChatGPT
- Midjourney
- DALL-E
- Ideogram
- Canva

## Level

intermediate

## Variables

- `[TOPIC]`
- `[AUDIENCE]`
- `[FORMAT]`
- `[BRAND_STYLE]`
- `[LANGUAGE]`

## Prompt

```text
Create a premium LinkedIn infographic concept about [TOPIC] for [AUDIENCE]. Format: [FORMAT]. Brand style: [BRAND_STYLE]. Language: [LANGUAGE]. Design it with strong hierarchy, concise copy, professional spacing, and a clear takeaway. Include headline, sections, icon suggestions, visual layout, and final caption idea.
```

## Example Use

```text
Create a premium LinkedIn infographic concept about the top 5 project control signals for infrastructure projects.
```

## Expected Output

A ready-to-design infographic brief with headline, layout, copy blocks, and visual direction.

## Notes

For image generators, keep text minimal and add final copy manually if the tool struggles with typography.
