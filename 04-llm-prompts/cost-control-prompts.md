# Cost Control Prompts

## Category

llm-prompts

## Recommended Use

Analyze project cost performance, variances, forecasts, and corrective actions.

## Ideal For

- Cost reports
- EVM review
- Forecasting
- Executive summaries

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

advanced

## Variables

- `[PROJECT_CONTEXT]`
- `[BUDGET_DATA]`
- `[ACTUAL_COSTS]`
- `[VARIANCES]`
- `[REPORTING_PERIOD]`

## Prompt

```text
Analyze the cost control status for this project. Project context: [PROJECT_CONTEXT]. Budget data: [BUDGET_DATA]. Actual costs: [ACTUAL_COSTS]. Variances: [VARIANCES]. Reporting period: [REPORTING_PERIOD]. Identify key cost drivers, explain variances, assess forecast risk, recommend corrective actions, and write an executive summary with clear financial implications.
```

## Example Use

```text
Analyze the cost control status for a transmission line project with procurement overruns and delayed civil works.
```

## Expected Output

A structured cost analysis with drivers, risks, forecasts, and corrective actions.

## Notes

Provide tables or exported report data for stronger analysis.
