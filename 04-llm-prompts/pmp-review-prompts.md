# PMP Review Prompts

## Category

llm-prompts

## Recommended Use

Review PMP topics with practical explanations, examples, questions, and memory aids.

## Ideal For

- PMP exam preparation
- Knowledge area review
- Scenario practice
- Agile/hybrid review

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

intermediate

## Variables

- `[TOPIC]`
- `[EXAM_DOMAIN]`
- `[DIFFICULTY]`
- `[LANGUAGE]`

## Prompt

```text
Help me review the PMP topic [TOPIC] from the [EXAM_DOMAIN] domain. Difficulty: [DIFFICULTY]. Language: [LANGUAGE]. Explain the concept, show a practical project example, list common traps, create 5 scenario-based questions, provide answers with explanations, and finish with a short memory aid.
```

## Example Use

```text
Help me review stakeholder engagement for the PMP exam with intermediate difficulty.
```

## Expected Output

A study-ready review with explanations, examples, quiz questions, and answer rationale.

## Notes

For exam prep, ask for scenario questions instead of definition-only questions.
