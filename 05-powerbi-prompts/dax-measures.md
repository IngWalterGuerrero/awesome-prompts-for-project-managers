# DAX Measures

## Category

powerbi-prompts

## Recommended Use

Generate, review, or improve DAX measures for Power BI reports.

## Ideal For

- KPI measures
- Time intelligence
- Cost performance
- Schedule analytics

## Compatible Tools

- ChatGPT
- Claude
- Gemini
- Power BI

## Level

advanced

## Variables

- `[BUSINESS_QUESTION]`
- `[TABLES]`
- `[COLUMNS]`
- `[FILTER_CONTEXT]`
- `[EXPECTED_RESULT]`

## Prompt

```text
Create a DAX measure for this Power BI requirement. Business question: [BUSINESS_QUESTION]. Tables: [TABLES]. Columns: [COLUMNS]. Filter context: [FILTER_CONTEXT]. Expected result: [EXPECTED_RESULT]. Provide the DAX measure, explain how it works, mention assumptions, and suggest validation checks.
```

## Example Use

```text
Create a DAX measure for cost variance percentage by project and reporting month.
```

## Expected Output

A DAX measure with explanation, assumptions, and validation guidance.

## Notes

Always provide model relationships and granularity for better DAX.
