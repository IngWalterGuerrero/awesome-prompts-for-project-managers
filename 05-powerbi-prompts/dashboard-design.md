# Dashboard Design

## Category

powerbi-prompts

## Recommended Use

Design an effective Power BI dashboard before building visuals.

## Ideal For

- Executive dashboards
- PMO reports
- Cost and schedule dashboards
- Portfolio views

## Compatible Tools

- ChatGPT
- Claude
- Gemini
- Power BI

## Level

intermediate

## Variables

- `[DASHBOARD_GOAL]`
- `[AUDIENCE]`
- `[KPIS]`
- `[DATA_SOURCES]`
- `[DECISIONS_SUPPORTED]`

## Prompt

```text
Design a Power BI dashboard for [DASHBOARD_GOAL]. Audience: [AUDIENCE]. KPIs: [KPIS]. Data sources: [DATA_SOURCES]. Decisions supported: [DECISIONS_SUPPORTED]. Recommend page structure, KPI hierarchy, visual types, slicers, drill-throughs, color logic, and narrative flow. Prioritize decision-making over decoration.
```

## Example Use

```text
Design a Power BI dashboard for monthly project portfolio performance.
```

## Expected Output

A dashboard blueprint with pages, visuals, filters, and design rationale.

## Notes

Start with decisions and questions, then choose visuals.
