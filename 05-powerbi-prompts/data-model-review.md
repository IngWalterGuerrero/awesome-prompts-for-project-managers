# Data Model Review

## Category

powerbi-prompts

## Recommended Use

Review a Power BI semantic model for clarity, performance, and analytical usefulness.

## Ideal For

- Star schema review
- Relationship checks
- Measure organization
- Performance improvement

## Compatible Tools

- ChatGPT
- Claude
- Gemini
- Power BI

## Level

advanced

## Variables

- `[MODEL_DESCRIPTION]`
- `[FACT_TABLES]`
- `[DIMENSION_TABLES]`
- `[RELATIONSHIPS]`
- `[KNOWN_ISSUES]`

## Prompt

```text
Review this Power BI data model. Model description: [MODEL_DESCRIPTION]. Fact tables: [FACT_TABLES]. Dimension tables: [DIMENSION_TABLES]. Relationships: [RELATIONSHIPS]. Known issues: [KNOWN_ISSUES]. Identify modeling risks, relationship problems, ambiguity, performance issues, missing dimensions, and opportunities to improve the model. Recommend a cleaner structure.
```

## Example Use

```text
Review a Power BI model with project costs, schedule milestones, vendors, and monthly forecasts.
```

## Expected Output

A practical model review with risks, recommended structure, and improvement priorities.

## Notes

Screenshots or table schemas improve the quality of the review.
