# Anti-Gravity / Living Artifact Presentation

## Category

Slide prompts / Executive presentations

## Recommended Use

Premium presentations for NotebookLM, Kael, Gamma, or AI slide generation tools when the deck should feel cinematic, intelligent, and executive-ready instead of generic or template-driven.

## Ideal For

- Executive presentations
- Project management
- Artificial intelligence
- Dashboards
- Innovation
- Project portfolios

## Compatible Tools

- NotebookLM
- Kael
- Gamma
- ChatGPT
- Claude

## Level

Advanced

## Variables

- `[TOPIC]`
- `[AUDIENCE]`
- `[NUMBER_OF_SLIDES]`
- `[STYLE]`
- `[LANGUAGE]`

## Prompt

```text
Create a premium presentation that feels like a living artifact, not a traditional slide deck.

Topic: [TOPIC]
Audience: [AUDIENCE]
Number of slides: [NUMBER_OF_SLIDES]
Visual style: [STYLE]
Language: [LANGUAGE]

Build a cinematic, executive-level narrative with a strong opening, clear strategic tension, elegant section transitions, and a memorable closing. Each slide should feel intentional, minimal, and visually premium. Avoid generic corporate layouts, dense bullet slides, decorative filler, and stock-photo thinking.

For each slide, provide:
- Slide title
- Core message
- Suggested visual direction
- Key talking points
- Speaker notes

Use concise language, strong hierarchy, and a sense of motion across the deck. The final result should feel like a premium strategic artifact that could be shown to a board, a PMO, or an innovation leadership team.
```

## Example Use

```text
Create a premium presentation that feels like a living artifact, not a traditional slide deck.

Topic: AI-driven project controls for infrastructure projects
Audience: executive board
Number of slides: 12
Visual style: minimal premium, dark editorial, high-contrast data storytelling
Language: English
```

## Expected Output

An executive presentation outline with a premium visual concept, clear narrative arc, slide-by-slide structure, and speaker-ready notes.

## Notes

Works best when the audience, decision context, and desired level of detail are clearly provided. If the result feels too abstract, add concrete business data, project constraints, and the decision the audience must make.
