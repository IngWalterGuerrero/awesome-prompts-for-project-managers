# Minimal Premium Deck

## Category

slide-prompts

## Recommended Use

Create a refined presentation with minimal layouts, sharp messaging, and polished executive structure.

## Ideal For

- Business proposals
- PMO updates
- Executive summaries
- Strategy narratives

## Compatible Tools

- Gamma
- Kael
- ChatGPT
- Claude

## Level

intermediate

## Variables

- `[TOPIC]`
- `[AUDIENCE]`
- `[NUMBER_OF_SLIDES]`
- `[TONE]`
- `[LANGUAGE]`

## Prompt

```text
Create a minimal premium deck about [TOPIC] for [AUDIENCE]. Use [NUMBER_OF_SLIDES] slides, a [TONE] tone, and write in [LANGUAGE]. Make the deck concise, elegant, and executive-ready. Each slide should have one clear idea, a short title, a tight message, and a suggested visual treatment. Avoid clutter, filler text, and generic stock-photo concepts.
```

## Example Use

```text
Create a minimal premium deck about project recovery strategy for a steering committee.
```

## Expected Output

A clean slide-by-slide deck plan with strong hierarchy and concise executive messaging.

## Notes

Add company context or project constraints for a more useful deck.
