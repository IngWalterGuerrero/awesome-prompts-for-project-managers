# Executive Board Presentation

## Category

slide-prompts

## Recommended Use

Prepare a board-level deck focused on decisions, risks, financial impact, and strategic options.

## Ideal For

- Steering committees
- Board updates
- Investment decisions
- Portfolio reviews

## Compatible Tools

- ChatGPT
- Claude
- Gamma
- Kael

## Level

advanced

## Variables

- `[DECISION]`
- `[CONTEXT]`
- `[OPTIONS]`
- `[RISKS]`
- `[RECOMMENDATION]`
- `[LANGUAGE]`

## Prompt

```text
Create an executive board presentation for the following decision: [DECISION]. Context: [CONTEXT]. Options: [OPTIONS]. Risks: [RISKS]. Recommendation: [RECOMMENDATION]. Language: [LANGUAGE]. Structure the deck for senior decision-makers. Lead with the decision needed, explain the strategic context, compare options, quantify impact where possible, show key risks, and close with a clear recommendation and next steps.
```

## Example Use

```text
Create an executive board presentation for deciding whether to accelerate a delayed infrastructure project.
```

## Expected Output

A board-ready deck structure with decision framing, option comparison, risk analysis, and recommended action.

## Notes

Include financial values, dates, and constraints whenever available.
