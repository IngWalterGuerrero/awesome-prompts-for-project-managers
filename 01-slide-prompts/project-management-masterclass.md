# Project Management Masterclass

## Category

slide-prompts

## Recommended Use

Build a structured masterclass presentation for project management concepts, frameworks, and practical examples.

## Ideal For

- PM training
- Internal workshops
- PMP review sessions
- Leadership development

## Compatible Tools

- NotebookLM
- Gamma
- ChatGPT
- Claude

## Level

intermediate

## Variables

- `[TOPIC]`
- `[AUDIENCE_LEVEL]`
- `[DURATION]`
- `[INDUSTRY]`
- `[LANGUAGE]`

## Prompt

```text
Create a project management masterclass about [TOPIC] for [AUDIENCE_LEVEL] participants in the [INDUSTRY] industry. Duration: [DURATION]. Language: [LANGUAGE]. Structure the session as a clear learning journey with modules, practical examples, exercises, reflection questions, and a final action checklist. Make the content practical, not academic.
```

## Example Use

```text
Create a project management masterclass about risk management for intermediate construction project managers.
```

## Expected Output

A training deck structure with modules, learning objectives, examples, exercises, and takeaways.

## Notes

Works well when paired with real project scenarios or lessons learned.
