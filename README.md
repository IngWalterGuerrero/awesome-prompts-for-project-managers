# Walter AI Prompt Library

A curated library of AI prompts for project managers, construction professionals, Power BI creators, and AI-powered business workflows.

This repository is designed as a practical prompt catalog: each prompt includes its recommended use, compatible tools, variables, expected output, and notes for adaptation.

## Categories

| Category | Description |
|---|---|
| [01 Slide Prompts](01-slide-prompts/) | Premium decks, executive presentations, NotebookLM/Kael/Gamma workflows, and narrative slide generation. |
| [02 Image Prompts](02-image-prompts/) | LinkedIn visuals, infographics, dashboard concepts, construction visuals, and premium image styles. |
| [03 Video Prompts](03-video-prompts/) | Reels, YouTube intros, cinematic transitions, documentary-style prompts, and short-form content. |
| [04 LLM Prompts](04-llm-prompts/) | ChatGPT, Claude, Gemini, and general-purpose assistant prompts for analysis, review, and ideation. |
| [05 Power BI Prompts](05-powerbi-prompts/) | DAX, dashboard design, data modeling, storytelling, and executive analytics workflows. |
| [06 Construction Projects](06-construction-projects/) | Substations, transmission lines, civil works, risk control, and field/project controls prompts. |
| [07 Project Management](07-project-management/) | Stakeholders, schedules, costs, risks, communications, lessons learned, and PM operating rhythms. |
| [08 Business Strategy](08-business-strategy/) | Executive thinking, business models, positioning, decision memos, and strategy narratives. |

## Prompt Standard

Every prompt should follow the shared structure in [prompt-template.md](prompt-template.md):

- Metadata for searchability
- Recommended use
- Ideal scenarios
- Compatible tools
- Variables
- Prompt text
- Example use
- Expected output
- Notes and adaptation guidance

## Suggested Workflow

1. Pick the category closest to the outcome you need.
2. Open the prompt file and replace variables such as `[TOPIC]`, `[AUDIENCE]`, `[STYLE]`, or `[LANGUAGE]`.
3. Run the prompt in the compatible AI tool.
4. Save improved versions as new files or propose changes through a pull request.
