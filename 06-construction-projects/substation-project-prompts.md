# Substation Project Prompts

## Category

construction-projects

## Recommended Use

Analyze, plan, and communicate substation project execution topics.

## Ideal For

- Engineering coordination
- Procurement follow-up
- Construction progress
- Commissioning readiness

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

advanced

## Variables

- `[PROJECT_PHASE]`
- `[VOLTAGE_LEVEL]`
- `[MAIN_CONSTRAINTS]`
- `[STAKEHOLDERS]`
- `[OUTPUT_FORMAT]`

## Prompt

```text
Act as a senior substation project manager. Project phase: [PROJECT_PHASE]. Voltage level: [VOLTAGE_LEVEL]. Main constraints: [MAIN_CONSTRAINTS]. Stakeholders: [STAKEHOLDERS]. Output format: [OUTPUT_FORMAT]. Analyze the situation, identify critical risks, define immediate actions, assign ownership, and prepare an executive-ready summary.
```

## Example Use

```text
Act as a senior substation project manager during commissioning readiness review for a 230 kV substation.
```

## Expected Output

A structured substation project analysis with risks, actions, owners, and executive summary.

## Notes

Include equipment status, energization date, and pending tests when possible.
