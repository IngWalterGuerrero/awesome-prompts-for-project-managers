# Transmission Line Prompts

## Category

construction-projects

## Recommended Use

Support planning, risk review, and execution control for transmission line projects.

## Ideal For

- Right-of-way issues
- Tower erection
- Stringing activities
- Environmental and access constraints

## Compatible Tools

- ChatGPT
- Claude
- Gemini

## Level

advanced

## Variables

- `[LINE_LENGTH]`
- `[VOLTAGE_LEVEL]`
- `[CURRENT_STATUS]`
- `[BLOCKERS]`
- `[TARGET_MILESTONE]`

## Prompt

```text
Act as a senior transmission line project controls advisor. Line length: [LINE_LENGTH]. Voltage level: [VOLTAGE_LEVEL]. Current status: [CURRENT_STATUS]. Blockers: [BLOCKERS]. Target milestone: [TARGET_MILESTONE]. Assess execution risk, identify schedule threats, recommend recovery actions, and prepare a stakeholder update.
```

## Example Use

```text
Assess execution risk for a 115 kV transmission line with right-of-way delays and tower foundation issues.
```

## Expected Output

A practical execution review with risks, recovery actions, and communication points.

## Notes

Separate technical, access, environmental, and stakeholder blockers.
